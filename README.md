# GUI-Agent-Demo
基于多模态大模型的简易GUI智能交互Agent，用于小游戏与界面自动化操作。

## 项目简介
本项目旨在构建轻量化端到端GUI Agent，实现从界面截图感知、多模态理解到自动化操作的完整流程，可适配简单小游戏与APP界面的交互任务。

## 技术栈
- Python 3.10+
- PyTorch
- Qwen-VL 多模态模型
- PyAutoGUI 自动化工具

## 项目进度
- ✅ 界面截图与元素解析模块（基础版本）
- ✅ 自动化操作控制脚本（支持点击、滑动）
- ⏳ 多模态决策逻辑优化（开发中）
- ⏳ 小游戏适配与鲁棒性测试（开发中）

## 使用说明
1. 安装依赖：`pip install -r requirements.txt`
2. 运行主程序：`python gui_agent.py`
